# Iris Flower Classification

This project builds a machine learning model to classify Iris flower species (Setosa, Versicolor, Virginica) using four features: sepal length, sepal width, petal length, and petal width. It includes preprocessing, scaling, training, evaluation, and visualizations.

**Features**
- Data preprocessing and cleaning
- StandardScaler for feature scaling
- Models: Logistic Regression, SVC, Random Forest
- Evaluation: Accuracy, Precision, Recall, F1-score, Confusion Matrix
- Visualizations: PCA scatter plot, ROC Curve

**Tech Stack**
Python, NumPy, Pandas, Scikit-learn, Matplotlib

**Files Included**
iris_classification.ipynb  
iris_classification.py  
Iris.csv  
README.md

**How to Run**
1. Install dependencies: pip install numpy pandas scikit-learn matplotlib  
2. Open the notebook or run the script  
3. Execute all steps to train and test the model

**Result**
The model accurately predicts Iris species and provides visual insights into class separability.
